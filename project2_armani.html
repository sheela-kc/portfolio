Armani Clothing Website

Test Cases

TC01: Login Functionality

Description: Verify that users can log in with valid credentials

Preconditions: User has created a registered account

Test Steps:

  1. Navigate to Armani's login page
  2. Enter a valid username
  3. Enter a valid password
  4. Click the "Login" button
  5. Verify that the user is redirected to the dashboard

Expected Result: User successfully logs in and sees the dashboard

TC02: Invalid Login Attempt

Description: Verify that an error message appears for invalid login attempts.

Preconditions: User account must exist

Test Steps:

  1. Navigate to the Southwest Armani's login page
  2. Enter an invalid username or incorrect password
  3. Click the "Login" button
  4. Verify that an appropriate error message is displayed

Expected Result: Error message appears, and login is denied

TC03: Clothing Item Search Functionality

Description: Verify that users can search for flights based on different criteria.

Preconditions: Website is accessible

Test Steps:

  1. Navigate to Armani's homepage
  2. Enter departure and arrival cities
  3. Select travel dates
  4. Choose the number of passengers
  5. Click 'Search' button
  6. Verify that search results display available flights

Expected Result: Relevant flight options appear with correct pricing.
